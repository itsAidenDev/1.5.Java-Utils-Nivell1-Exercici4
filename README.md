"Add the functionality to read any TXT file and display its contents by console".

The goal is to gain experience in file reading, handling file I/O exceptions, and displaying file content, which are useful skills for tasks like processing configuration files, logs, or any other data stored in text format. This exercise also reinforces the importance of resource management and exception handling in file-based applications.
